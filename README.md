#  Простой корзина товаров на React

Это учебный проект, демонстрирующий базовые концепции React:  
- Компоненты  
- Управление состоянием (`useState`)  
- Передача функций через props  
- Отображение динамических данных

---

##  Что умеет приложение

- Отображает список товаров (iPhone, MacBook, AppleTV)
- Каждый товар имеет кнопку «Добавить»
- Счётчик общего количества добавленных товаров обновляется в реальном времени

---

##  Как использовать

1. Кликни по кнопке **«Добавить»** рядом с любым товаром
2. Наблюдай, как увеличивается счётчик внизу страницы

---

##  Как это работает

- В `App.jsx` хранится общее состояние `count`
- Функция `handleClick` увеличивает счётчик
- Каждому компоненту `Product` передаётся:
  - `name` — название товара
  - `onAdd` — функция для увеличения счётчика

---

##  Замечание

Это **учебный проект** — данные не сохраняются между перезагрузками.  
В будущем можно добавить:
- Сохранение в `localStorage`
- Удаление товаров
- Сумму стоимости
